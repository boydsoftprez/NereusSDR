Phase mapping: Alex-2 HPF + LPF edge editors + master bypass = B · Live LED active-filter indicators (poll from ep6 status / I/O state) = H
Spin from Thetis: NereusSDR uses CSS-glow LEDs to make active filters scannable at a glance; Thetis uses solid-color LED bitmaps. Same semantics.
Active bar: shows board-detected status + current HPF/LPF live; auto-hides this whole tab when board model has no Alex-2 (e.g., HL2, Hermes).
Persistence: per-MAC under hardware/<mac>/alex2/{hpf,lpf}/<band>/{enabled,start,end}.

Two columns: HPF (with master bypass + per-band edge editors) · LPF (no master toggles). LED rows above each table indicate the live filter selection. Look right?
