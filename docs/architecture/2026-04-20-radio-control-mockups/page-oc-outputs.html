Phase mapping: RX/TX matrix + master toggles + TX pin action + USB BCD + ext PA + reset = D (OcMatrix port from HPSDR/Penny.cs:33-150 [ramdor 501e3f5] + setup.cs:12638-12760) · Live OC pin state LED row = H
Spin from Thetis: NereusSDR has 14 bands vs Thetis's 12 — GEN/WWV rows are disabled by default (no OC sense for "all-mode receive" or "WWV"); XVTR enabled. Color-coded matrix (blue=RX, red=TX) makes the two grids scannable; Thetis uses identical-looking checkboxes.
Persistence: per-MAC under hardware/<mac>/oc/{rx,tx}/<band>/pin<n> + .../actions/pin<n>/{mox,tune,2tone,vox,pain}.

All four Thetis-shape sub-areas on one tab: master toggles · RX matrix (blue) + TX matrix (red) · TX pin action mapping · USB BCD + ext PA + live state. Heavy but matches Thetis 1:1.
